# Van Horn Reborn

## 1. Installation
spooni_spooni_van_reborn and spooni_van_mutant_mansion works Standalone.  

To install spooni_spooni_van_reborn and spooni_van_mutant_mansion:
- Drag and drop the resource into your resources folder
  - `spooni_spooni_van_reborn`
  - `spooni_van_mutant_mansion`
- Add this ensure in your server.cfg
  - `ensure spooni_spooni_van_reborn`
  - `ensure spooni_van_mutant_mansion`
- At the end
  - Restart the server
