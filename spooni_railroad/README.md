# spooni_railroad

## 1. Installation
spooni_railroad works Standalone.  

To install spooni_railroad:
- Drag and drop the resource into your resources folder
  - `spooni_railroad`
- Add this ensure in your server.cfg
  - `ensure spooni_railroad`
- At the end
  - Restart the server

If you have any problems, you can always open a ticket in the 
[Spooni Discord] https://discord.gg/spooni
