# spooni_bla_reborn2

## 1. Installation
spooni_bla_reborn2 works Standalone.  

To install spooni_bla_reborn2:
- Drag and drop the resource into your resources folder
  - `spooni_bla_reborn2_1`
  - `spooni_bla_reborn2_2`
- Add this ensure in your server.cfg
  - `ensure spooni_bla_reborn2_1`
  - `ensure spooni_bla_reborn2_2`
- At the end
  - Restart the server

If you have any problems, you can always open a ticket in the 
[Spooni Discord] https://discord.gg/spooni
