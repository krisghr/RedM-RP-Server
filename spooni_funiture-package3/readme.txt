Welcome to Spooni Funiture package for RDR2. 
---------------------------------------
We actually create any prop or houses you want. 

Ask us on Discord https://discord.gg/spooni.
-------------------------------------------------------

How is it installed?

Copy spooni_funiture-package3 into the resources folder. 

Then in the server.cfg
write

ensure spooni_funiture-package3


---------------------------------------------------------------------------
In this Package you have more as 50 new Props there you can use in Spooner.

Download the new Spooni Spooner here to easy use the Props Houses and Stables

https://github.com/Spooni-Development/spooni_spooner

then restart server. 

Start Spooner under Spooni, you see the list of Spooni Houses, Stables and Props is now at the top. 

It is important that all objects are in it, but of course only those that you have purchased will work.


Creat a doorhash for all doors.

have fun.
